# Resume Placeholder

## To Add Your Resume:

1. Create a one-page PDF resume named `Boden_Crouch_Resume.pdf`
2. Place it in the `/public` directory
3. The hero section already links to `/Boden_Crouch_Resume.pdf`

## Resume Tips:

- Keep it to one page
- Include:
  - Contact information (email, GitHub, LinkedIn)
  - Professional summary (2-3 sentences)
  - Key technical skills
  - Work experience with metrics
  - Notable projects with outcomes
  - Education/certifications

## Quick Generation Tools:

- LaTeX with moderncv
- Canva (professional templates)
- Figma
- Google Docs → Export as PDF
